# MUVIRP Website

Proyecto de sitio web para MUVIRP.

## Estructura del Proyecto

```
muvirp/
├── index.html          # Página principal
├── styles.css          # Estilos generales
├── script/
│   └── scripts.js      # Scripts principales
├── shop/
│   ├── od/
│   │   ├── delictivo.css
│   │   └── delictivo.html
│   └── vehicles/
│       ├── auto.css
│       └── autos.html
└── style/
    ├── styles.css
    └── img/            # Imágenes
```

## Descripción

Este proyecto contiene el sitio web de MUVIRP con secciones para shop, vehículos y contenido delictivo.

## Cómo usar

1. Abre `index.html` en tu navegador
2. Navega a través de las diferentes secciones
